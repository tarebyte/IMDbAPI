# Imdbapi

This is a unofficial IMDB Ruby wrapper for getting all of the wonderful
movie data that everybody scrapes for.

## Installation

Add this line to your application's Gemfile:

    gem 'imdbapi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install imdbapi
